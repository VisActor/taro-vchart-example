# @visactor/taro-vchart 使用示例

## 如何使用

1. 安装依赖

```bash
$ yarn
```

2. 启动编译，根据你想要体验的环境运行对应的命令

```bash
# h5 环境，启动后浏览器会自动打开
$ yarn dev:h5

# 飞书小程序，启动后通过飞书开发者工具导入编译后的 dist 目录即可
$ yarn dev:tt

# 微信小程序，启动后通过微信开发者工具导入编译后的 dist 目录即可
$ yarn dev:weapp
```
